# 🧪 Ручные тесты: Соискатель

## Профиль

| № | Сценарий | Шаги | Ожидаемый результат |
|---|----------|------|-------------------|
| 1 | Просмотр профиля | Зайти в "Мой профиль" | ✅ Отображаются все данные из БД |
| 2 | Редактирование | Изменить город, зарплату, "О себе" → Сохранить | ✅ Данные обновлены, в БД записаны новые значения |
| 3 | Смена профессии | Выбрать другую профессию из справочника | ✅ Профессия обновилась, в резюме отобразилась новая |

## Резюме

| № | Сценарий | Шаги | Ожидаемый результат |
|---|----------|------|-------------------|
| 4 | Создание резюме | Заполнить форму → "Создать" | ✅ Резюме появилось в списке, статус "на модерации" |
| 5 | Просмотр списка | Перейти в "Мои резюме" | ✅ Список резюме с заголовками, зарплатами, статусами |
| 6 | Активация/деактивация | Переключить тумблер "Активно" | ✅ Статус изменился, резюме скрыто/показано в поиске |

## Отклики

| № | Сценарий | Шаги | Ожидаемый результат |
|---|----------|------|-------------------|
| 7 | Отклик на вакансию | Нажать "Откликнуться" → написать сопроводительное | ✅ Отклик создан, статус "pending" |
